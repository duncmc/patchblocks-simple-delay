Simple Delay
============

A simple delay patch with feedback level for [Patchblocks](http://patchblocks.com).

Controls
--------
Left Knob: Delay time  
Right Knob: Feedback level  
Left Button: Effect on/off  

Left LED indicates effect on/off.
